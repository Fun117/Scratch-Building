# ガイド / MOD
データに保存されているMODの情報を解析、表示することができる。データベースは現在実装されてない為、オブジェクト変数にデータを保存している。

## MODの情報を表示する
```html
<div id="mods"></div>
```
> 上のコードの様に```id=mods```の要素にMOD情報が追加される。

## TAGIDの説明
```tagId```には、```0``` ```1``` ```2```があります。
- ```0``` / ```一般```（tagが表示されることはない）
- ```1``` / ```公式```（tagが表示され```name```がタグの名前として表示される。）
- ```2``` / ```前提```（tagが表示され```name```がタグの名前として表示される。）
