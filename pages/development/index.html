ウェブサイトの移行

新しいサイトに移行しました！移行をした為、このサイトの一部機能が利用できなくなっている可能性があります。移行したサイトの開発を優先的に行う為、旧サイトは修正等が遅れる場合があります。

開発ページ

公開日：2023-12-24
最終更新：2023-12-24

API

APIはMOD情報や更新履歴を取得する事ができる。

APIの対応状況

  • 更新履歴
  • MOD情報
  • よくある質問
