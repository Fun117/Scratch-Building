SB MODの使い方

公開日：2023-12-15
最終更新：2023-12-15


MODページを見る

初めに

SB MODとは、ビル経営ゲームv7でMODを作成する時に役に立つツールである。
スプライトを好きなビル経営ゲームクライアントに追加することで自分の好きな要素を追加できる。

導入方法

こちらのプロジェクトに "SB MOD" , "SB MOD GUI" というスプライトがある。
スプライトは本体と、アドオン用のON/OFFを切り替えられる設定画面を出すものであり、ビル経営ゲームのデータを操作したい場合はSB MODを、設定画面なども作りたい場合は2つのスプライトを使う。

メソッド - set

このメソッドでは、ゲーム内のデータを操作できる。

gold

所持金を設定する

SB MOD[set][gold][12000][]::#ff6680 //所持金を12000Gにする

happiness

幸福度を設定する

SB MOD[set][happiness][100][]::#ff6680 //幸福度を100%にする

diamond

ダイヤモンドを設定する

level

レベルを設定する

xp

XPを設定する

x

カメラのx座標を設定する

y

カメラのy座標を設定する

day

経過日数を設定する

time

ゲーム内時間を設定する

weather

天候を設定する

0:晴れ 1:雨 2:大雨

雨か雪かは地域に依存します。

background

背景が曇りかを設定する

0:晴れ 1:曇り

map

地域を設定する

0:平原 1:雪山 2:砂漠

tablet [SB-page]

タブレット画面関連を設定する

SB MOD[set][tablet][tab][2]::custom //タブレットのタブを2にする

setting [SB-page]

設定画面関連を設定する

SB MOD[set][tablet][page][addon_setting]::custom //設定画面のページをaddon_settingにする

cloudRequest
高度

クラウドの戻り値を変更する

SB MOD[set][cloudRequest][(("user",(0,0,0,0,0)))][]::custom

これを実行することにより、ビルハブなどのクラウド通信を使う場所で戻り値を変えることができる。

メソッド - post

cat

猫に関する動作

spawn: 猫をスポーンする

despawn: 全ての猫をデスポーンさせる

SB MOD[post][cat][spawn][]::custom SB MOD[post][cat][despawn][]::custom

mouse

マウスに関する動作

lock: マウスを無効化

unlock: 無効化を解除

※値がlockではない場合は解除となる。

SB MOD[post][mouse][lock][]::custom

toast

トーストを表示する

toast [コスチューム] [効果音]

SB MOD[post][toast][3][1]::custom

dialog

ダイアログを表示して待つ

dialog [コスチューム] [ボタン]

マウスをロックしてから実行することを推奨します。

戻り値は(SBmod: return)に表示される。

SB MOD[post][mouse][lock][]::custom SB MOD[post][dialog][4][3]::custom //ダイアログの例 もし<(SBmod: return)=[-1]>なら{ x を押されたときの処理::grey }::control もし<(SBmod: return)=[0]>なら{ はい を押されたときの処理::grey }::control もし<(SBmod: return)=[1]>なら{ いいえ を押されたときの処理::grey }::control SB MOD[post][mouse][unlock][]::custom

ads

広告を表示する

マウスをロックしてから実行することを推奨します。
